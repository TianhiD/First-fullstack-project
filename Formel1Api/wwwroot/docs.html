This endpoint allows you to do get the information from the F1DB database in JSON format in the web-view. Below you will se examples on how to get the data with the use of endpoints, and a result image

General endpoints

Base url: http://localhost:5210/

GET url: http://localhost:5210/api/

Drivers url: http://localhost:5210/api/drivers

Teams url: http://localhost:5210/api/teams

Races url: http://localhost:5210/api/races

GET by ID:

Drivers url: http://localhost:5210/api/drivers/{id}

Teams url: http://localhost:5210/api/teams/{id}

Races url: http://localhost:5210/api/races/{id}

IMAGE Upload

Image url: http://localhost:5210/api/ImageUpload
